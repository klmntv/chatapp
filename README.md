Я постарался сделать проект максимально похожим на рабочую среду ChatApp: React + TypeScript, Redux Toolkit, React Hook Form, Vite, ESLint + Prettier, Ant Design, SCSS Modules.
### Что внутри

- Полноэкранный layout: `nav` (панель) + `main` (контент).
- Фильтры задач в сайдбаре + счетчики `Все / Активные / Выполненные`.
- DND в списке задач.
- Выполненные задачи автоматически группируются внизу.
- Минимальный адаптив: на ширине `<= 768px` панель принудительно свернута.
- `ErrorBoundary` на основной контент.
- FSD: `src/features` для фич и стейта, `src/widgets` для композиции UI-блоков.
